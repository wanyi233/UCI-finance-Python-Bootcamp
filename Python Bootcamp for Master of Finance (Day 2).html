1. Access financial data¶

Nowadays, millions of data are generated, transmitted and stored on Internet every day. Many financial institutions, government departments and tech companies offer public access to large amount of well-organized data. These data can be accessed through application programming interface (API), an interface that allow Python (and other programming languages) to download data directly within your code.

We will learn a cool package: pandas-datareader. Before that, let me introduce Python "package" to you.

1.1 Package in Python¶

What is package? Why do we need packages?

  • A python package is collection of functions that perform certain tasks.
  • They are ready-to-use tools so you dont need to build everything from the very beginning.
  • For example:
    • Programming: build a house
    • Function: hammer, screwdriver, nail, etc.
    • Package: a toolbox

1.1.1 Download and install package¶

Anaconda already pre-installed many common packages. However, if you want to use a package that is not on your computer yet, you need to download and install it manually. We can use the following command to install a package:

pip install package_name

Note that this is NOT a Python command, but Jupyter Notebook has a "magic" feature: if you put an exclamation mark before the command, you can execude the code within Jupyter Notebook.

In [ ]:

!pip install pandas-datareader

Note: You only need to install the package ONCE. You don't need to run the code every time you want to use the package.

1.1.2 Import package¶

To use Python packages, you need to import it into your code. Just like you need to take the hammer out of the toolbox if you want to use it.

There are several ways to import a package:

  • import package_name: import the whole package.
  • import package_name as abbr.: import the whole package and refer to it as an abbreviation.
  • from package_name import function_name: only import a specific function from the package.
In [1]:
# Example
import pandas_datareader
import pandas_datareader as pdr
from pandas_datareader import get_data_yahoo

Note:: You need to import the package EVERY TIME you execute the code.

1.2 Access data from Pandas-datareader¶

Pandas datareader is a remote data access package that allows one to get data from various internet data sources.

You can access data from:

  • Yahoo! finance: stock price, trading volume
  • Nasdaq Trader symbol: a list of all available equity symbols from Nasdaq.
  • St.Louis FED (FRED): Federal Reserve's economic data
  • Kenneth Frenchs data library: Fama-French factors, risk free rate, etc.
  • World Bank: gdp, international debt, human capital, etc.

Other data sources supported:

  • Tiingo, IEX, Alpha Vantage, Enigma, Quandl, Eurostat, Thrift Savings Plan, Stooq, MOEX

Each data source is accessed using different functions or parameters, because each API requires different information to be privided. Documentation on how to access each data source with pandas-datareader can be found here.

2. Analyze financial data¶

After downloading your data, you can now perform many meaningful financial analyses. Pandas is the workhorse to conduct data analysis in Python. It organizes data in a way similar to Excel, but augmented all the power and flexibility of Python. In this section, you will learn how to:

  • Create a table from scratch/a local file
  • Select a subset of the table
  • Modify the table to add more information
  • Calculate financial statistics from the table

Pandas stores data in "DataFrame". What is DataFrame?

  • A table-like data structure in Pandas.
  • Similar to "spreadsheet" in Excel.
  • DataFrame has rows and columns.
    • Each column has a name (e.g. "High", "Low", "Volume")
    • Each row has an index (e.g. "2015-12-07")

2.5 Calculate financial statistics¶

2.5.1 Convert data frequency¶

In finance, you often need to convert among different data frequencies. For example, daily to weekly, daily to monthly, etc.

Pandas has a ready-to-use function asfreq that can finish this task in one line! (super cool)

Syntax: New_variable_name = DataFrame_name.asfreq('frequency_to_convert').

In [34]:
amazon_month = amazon_price.asfreq('BM')  # get the stock price on each business month end
amazon_month.head(4)

Adj Close

Common time frequency abbreviations:

  • B: business day frequency
  • D: calendar day frequency
  • W: weekly frequency
  • M: month end frequency
  • Q: quarter end frequency
  • A: year end frequency
  • B[M/Q/A]: business month/quarter/year end frequency
  • [M/Q/A]S: month/quarter/year start frequency
  • B[M/Q/A]S: business month/quarter/year start frequency

Exercise 4: Visualize minimum variance frontier

This code below randomly generate 10,000 portfolios constructed of the five stocks and calculate the risk and return of each portfolio. Note that this chunk of code is beyond the scope of this BootCamp, but feel free to dig into it if you are interested.

In [50]:
import numpy as np
from random import uniform

# calculate the expected return and coviance of the five stocks
ret_mean = stock_returns.mean()
ret_cov = stock_returns.cov()

portfolio_list = []       # define an empty list

for i in range(10000):    # Generate one random portfolio in a loop. Iterate 10000 times. 
    
    # generate a set of random weight on 5 portfolios which add up to be 1. Allow negative weight.
    w = [uniform(-1,2), uniform(-1,2), uniform(-1,2), uniform(-1,2)]
    weight = w + [1 - sum(w)]

    # calculate portfolio return and standard deviation
    port_ret = sum(weight*ret_mean)
    port_std = np.sqrt(np.dot(weight, np.dot(ret_cov, weight)))
    
    # add the portfolio to list
    portfolio_list.append([port_ret, port_std] +  weight)

# create a dataframe from the list
portfolio = pd.DataFrame(portfolio_list, columns=['return', 'std', 'weight_AAPL', 'weight_CVX', 
                                                  'weight_SBUX','weight_NKE', 'weight_PFE'])
