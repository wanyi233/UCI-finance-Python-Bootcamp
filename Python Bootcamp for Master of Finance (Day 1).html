Variables are containers for storing data values:

variable name → noun = "cat" ← variable value

Benefit of using variables:

  • Do not need to make revisions one by one if we change the code.
  • Can use a lot of built-in functions.

Key points about variables:

  • Unlike other programming languages, Python has no command for declaring a variable.

  • A variable is created the moment you first assign a value to it.

  • Variables do not need to be declared with any particular type and can even change type after they have been set.

Variable Names:

  • A variable name cannot start with a number.
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ ).
  • Variable names are case-sensitive (age, AGE and and Age are three different variables).

A good habit: Use descriptive and clear variable names.

4.3 Loop: While loop¶

In python, while loop is used to execute a block of statements repeatedly until a given a condition is NOT satisfied. When the condition becomes false, the loop will be ended, and the line immediately after the loop in program will be executed.

In [30]:
i = 0
while i < 6:
    print("i =", i)
    i = i + 1
print("i =", i, "Loop finished!")

i = 0
i = 1
i = 2
i = 3
i = 4
i = 5
i = 6 Loop finished!

While loop can be pretty useful when you don't know how many times the loop will be executed.

In Python, break and continue statements can alter the flow of a normal loop.

Source: https://www.programiz.com/python-programming/break-continue

4.4.1 Break statement¶

The break statement terminates the loop containing it. Control of the program jumps to the code immediately after the loop.

In [31]:
# Example
i = 0

while 1:
    print(i)
    i = i + 1
    if i >= 6:
        break

print("Done!")

4.4.2 Continue statement¶

The continue statement is used to skip the rest of the code inside a loop for the current iteration only. Loop does not terminate but continues on with the next iteration.

In [32]:
# Example: print out numbers not divisible by 3
for i in range(6):      # range(n) = 0,1,2,...,n-1
    if i % 3 == 0:
        continue
    print(i)            # note that 0 and 3 are not printed

5.2 Dictionary¶

Python dictionary is an unordered collection of items. While other compound data types have only value as an element, a dictionary stores items in key-value pairs. A dictionary is defined by a pair of curly brackets.

In [48]:
# Use a pair of curly brackets to define a dictionary. Use colon to declare key-value pairs.
LuckyDict = {'Alex': 24, 
             'Betty': 9, 
             'Catherine': 7, 
             'David': 4,
             'Elizabeth': 2}
print(LuckyDict)

{'Alex': 24, 'Betty': 9, 'Catherine': 7, 'David': 4, 'Elizabeth': 2}

Important properties:

  • Values can be duplicated, but keys need to be unique.
  • values are mutable, but keys are immutable.
  • Dictionary is unordered.
In [49]:
# If keys are duplicated, the latter value will be memorized
LuckyDict2 = {'Alex': 24,

             'Catherine': 7, 
             'David': 4,
             'Alex': 2}
LuckyDict2
Out[49]:
{'Alex': 2, 'Betty': 9, 'Catherine': 7, 'David': 4}

# Dictionary is unordered
Dict1 = {'a': 2, 'b': 3, 'c': 5}
